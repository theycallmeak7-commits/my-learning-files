🧠 Principles · Mindset shifts

📈 Compound Learning & Pareto 80/20

Small daily improvements (1% better each day) double your ability in 70 days. Pareto Principle: 20% of efforts yield 80% of results — eliminate noise, focus on vital few.

Skill stackingCombine top-20% skills (coding + communication + business) to create unique value.
Failures don’t countOnly your last success defines you. Each failure is feedback, not a stain.

🎯 Choice vs Chore

Treat learning as a choice → intrinsic drive. Chores drain willpower; choices empower.

💡 It’s all in the frame

Obstacle = growth signal. Beginner mindset: “I’m not there yet” vs fixed attitude.

section 4

🔬 Neurobiology of learning · Science

🧠 Focus vs diffuse, sleep, myelin, spaced repetition, deliberate practice

Focus mode (laser) + diffuse mode (wandering) → back-and-forth builds myelin sheaths, boosting signal speed. Sleep washes toxins, consolidates memory. Spaced repetition flattens forgetting curve. Deliberate practice (edge of ability, immediate feedback, specific goals) unlocks breakthroughs.

📖 Passive vs Active learning

Passive (watching) feels easy but creates illusion of competence. Active recall & self-testing builds durable long-term memory.

⏳ Procrastination science

Triggered by emotional avoidance → start small (Pomodoro 25min). Starting reduces negative feeling within minutes.

⚙️ Techniques · Master tools

🍅 Pomodoro, Interleaving, Memory Palace, Einstellung, Habits, System vs Goals

Pomodoro: 25min focus + 5min diffuse → conquers procrastination. Interleaving: mix different skills (e.g., coding + design + speaking) to boost creativity. Einstellung effect: rigid mindset blocks solutions → always ask “what’s option C?”. Habits: cue-routine-reward loop, “don’t break the chain” method. System vs goals: systems (daily 20min) outperform rigid goals long-term.
